# Sistema de Ordem de Serviço

Este projeto é um sistema simples desenvolvido em **Python** que simula o registro e gerenciamento básico de uma **Ordem de Serviço (OS)**.

O objetivo do projeto é praticar conceitos fundamentais de programação, lógica e organização de código utilizando Python.

## Funcionalidades

* Registro de informações da ordem de serviço
* Validação de dados inseridos pelo usuário
* Organização do fluxo de atendimento
* Exibição das informações registradas no terminal

## Conceitos utilizados

* Entrada de dados com `input`
* Estruturas condicionais (`if / else`)
* Estruturas de repetição (`while`)
* Validação de dados
* Organização de lógica de programação

## Tecnologias

* Python

## Objetivo do projeto

Este projeto foi desenvolvido com fins educacionais, para praticar lógica de programação e consolidar conceitos básicos da linguagem Python durante os estudos.

## Como executar

1. Clone o repositório

```bash
git clone https://github.com/IvensRic/Projeto_Ordem_de_Servico.git
```

2. Entre na pasta do projeto

```bash
cd Projeto_Ordem_de_Servico
```

3. Execute o arquivo Python

```bash
python nome_do_arquivo.py
```
